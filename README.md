# trick-list-source

A simple and interactive web application for managing skateboard tricks.

## Features

- Add new skateboard tricks
- Mark tricks as completed (click on trick name)
- Delete tricks
- Data persists in browser's localStorage
- Clean and responsive design

## How to Use

1. Open `index.html` in a web browser
2. Enter a trick name in the input field
3. Click "Add Trick" or press Enter to add it to your list
4. Click on a trick name to mark it as completed
5. Click "Delete" to remove a trick from your list

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
